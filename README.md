# 프로젝트

### SKT TANGO 시스템 유지보수 개발
- SKT 통신망 관제 시스템의 공통파트(TANGO-PF)로 회원가입, 로그인, 서비스데스크, 공지사항, 사용자 관리, 협력업체 관리 등을 관리하는 파트입니다.   


<br/>

### 개발언어 / 라이브러리
- JDK 1.8 
- SVN 1.6.11
- JAVA / Spring 4.2.9 RELEASE
- Maven
- Redis 4.0.11   
- Oracle 12c
- Mybatis 

<br/>

### 운영업무 관리 화면 개발
- TANGO 시스템 운영에 관련 업무일지나 파일을 올려 관리자가 확인 및 승인 할 수 있는 화면 개발   

<br/>

### DB변경요청 화면 개발
- 배포 주차 별로 DB변동사항이 있을시에 각 파트 DB 변동내용과 파일을 올리고 DBA가 내용을 확인하여   
최종적으로 관리자의 승인후 작업을 진행하는 화면 개발   

<br/>

### 중복로그인 차단 기능 개발
- Redis를 이용하여 이미 로그인 되어있는 사용자가 다른 PC또는 환경에서 로그인을 시도하면   
기존에 로그인 되어 있던 세션이 끊기며 중복로그인을 차단하는 기능 개발
