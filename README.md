# 프로젝트

## SKT TANGO 시스템 유지보수 개발
- SKT 통신망 관제 시스템의 공통파트(TANGO-PF)로 회원가입, 로그인, 서비스데스크, 공지사항, 사용자 관리, 협력업체 관리 등을 관리하는 파트입니다.   


<br/>

### 개발언어 / 라이브러리
- JDK 1.8
- SVN 1.6.11
- JAVA / Spring 4.2.9
- Maven
- Oracle 12c
- Mybatis

<br/>

### 운영업무 관리 화면 개발
- TANGO 시스템 운영에 관련 업무일지나 파일을 올려 관리자가 확인 및 승인 할 수 있는 화면 개발   

<br/>

### DB변경요청 화면 개발
- 배포 주차 별로 DB변동사항이 있을시에 각 파트 DB 변동내용과 엑셀 파일을 올리고 DBA가 내용을 확인하여   
최종적으로 관리자의 승인후 작업 진행을 확인하는 화면 개발   

<br/>

### 계정잠금, 계정탈퇴 처리전 알림 전송 batch 개발
- 일정기간동안 로그인을 하지않은 사용자의 계정잠금, 계점탈퇴 처리전 해당되는 사용자에게 내부 Api를 이용하여   
mms와 메일을 전송하는 Spring batch 개발

### 상세업무
1. 공통   
Java, Spring Boot framework를 이용한 API 개발   
Spring JPA를 이용하여 MySql DB 연결 및 개발   
zenkins를 이용하여 프로젝트 빌드버전 관리,배포   
MSA 아키텍쳐를 이용하여 서비스를 분리하여 각각의 프로젝트로 개발   
eureka를 이용하여 다른 프로젝트와 데이터 연동 개발   
SVN을 이용하여 소스 버전 관리   
javaScript, jQuery 를 이용하여 서버단과 비동기 통신 기능 개발   

2. 세션관리   
Redis를 활용하여 로그인 세션 유지 및 다른 환경에서 동시에 로그인이 불가능 하도록 기능 개발   

3. 권한관리   
관리자의 사용자관리 기능을 만들어 사용자를 관리할 수 있는 기능 개발   
사용자의 권한에 따라 사용가능한 메뉴,화면을 부여하고 API 호출 시 체크하는 기능 개발
